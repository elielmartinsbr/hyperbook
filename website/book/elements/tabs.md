# Tabs

Tabs are a great way to display alternative content. In Hyperbook you
can use the `::::tabs` and `:::tab{A Tab Title}`.

```md
::::tabs

:::tab{Hi}
Here is a tab

Another Test
:::

:::tab{Huh}
Another tab with a [link](#)

Other

:::

:::tab{Third}

Wow

:::

::::
```
